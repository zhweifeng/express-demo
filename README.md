# express-demo
express 前端框架的一个demo  路由和接口的测试
前提是  安装node和git

### 全局安装nodemon

  
  `npm install -g nodemon`
  
  修改服务需要重启服务 nodemon模块可以自动重启服务  让我们只关心代码，在你修改代码保存后 nodemon 会自动重启服务 刷新页面即可
  
  如果不想安装可以修改package.json 里边的start的值  将nodemon改为node 即可

接着......

`npm install`


`npm start`
